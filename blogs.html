Question-3: What is event delegation in the DOM?
Event delegation is a method in the DOM that allows a parent element to manage events for its child elements through event bubbling. Rather than adding event listeners to each child element individually, you can attach a single listener to a shared ancestor.
Question-4: What is event bubbling in the DOM?
Event bubbling is a mechanism in the DOM where an event begins at the target element and then moves upward through its parent elements.
Question-5: How do you create, add, and remove elements using JavaScript?
To create an element, you can use document.createElement(). You can add it to the document using appendChild() or insertBefore(). To remove it, use removeChild() or remove().
